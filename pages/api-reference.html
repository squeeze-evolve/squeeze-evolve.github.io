CLI & Server API

squeeze-evolve-client

squeeze-evolve-client --config CONFIG --input INPUT [--output OUTPUT] [--n-problems N] [--include-path DIR]
FlagRequiredDescription
--configyesYAML, JSON, Python file, or registry name
--inputyes.parquet, .jsonl, or .json dataset
--outputnoOutput file (stdout if omitted)
--n-problemsnoLimit number of problems
--include-pathnoDirectory of .py config presets

Examples

# Basic run
squeeze-evolve-client \
  --config configs/examples/qwen3-235b_qwen3-30b_math.yaml \
  --input data/aime25/test.parquet

# Save results, limit to 5 problems
squeeze-evolve-client \
  --config configs/examples/three_model_math.yaml \
  --input data/aime25/test.parquet \
  --output results.json \
  --n-problems 5

POST /run

Execute a full evolutionary loop. Request body:

{
  "config": {
    "run_name": "my_run",
    "routing": { "k": 4, "population": 16, "loops": 5 },
    "models": [
      { "name": "cheap", "base_url": "http://localhost:8001/v1" },
      { "name": "expensive", "base_url": "http://localhost:8000/v1" }
    ]
  },
  "problems": [
    { "orig_prompt": "Find the sum of...", "gt": "42" }
  ]
}

Response:

{
  "run_id": "my_run_a1b2c3d4",
  "metrics": [ { "loop": 0, "..." } ],
  "problems": [ { "orig_prompt": "...", "candidates": ["..."] } ]
}
