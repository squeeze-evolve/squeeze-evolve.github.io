Running benchmarks

Single config

squeeze-evolve-client \
  --config benchmarks/aime25/qwen3-30b_235b.yaml \
  --input data/aime25/test.parquet \
  --output results/aime25_qwen3-30b_235b.json

Full suite

bash scripts/run_aime25.sh
bash scripts/run_hmmt25.sh
bash scripts/run_gpqa_diamond.sh
bash scripts/run_lcbv6.sh

Each script loops over all configs with 4 seeds (1234, 2345, 3456, 4567) and skips existing results.
